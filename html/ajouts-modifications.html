Ajouts et modifications

Gameplay

  • Mode Hardcore pour les Realms : Le mode hardcore est désormais activé pour les parties Realms.
  • Écran de création de monde mis à jour pour les Realms : Le processus de création de monde sur Realms bénéficie d’une interface revue.
  • Mises à jour des chambres d’essai (Trial Chambers) : Les trials chambers disposent maintenant de nouvelles variations.
  • Ajout de bébés versions des créatures marines : Les dauphins, poulpes et poulpes luisants ont désormais des versions bébé.
  • Variantes de saumon : De nouvelles variantes de saumon ont été ajoutées.
  • Modifications de la mécanique des Slime Blocks : Le fait de s’accroupir en rebondissant sur un bloc de slime n’annule plus les dégâts de chute.

Objets et entités

  • Perles d’Ender lancées : Les perles de l’End lancées chargent désormais les chunks.
  • Motifs de bannière : Les motifs de bannière ont reçu de nouveaux noms et icônes, avec l’ajout de deux nouveaux motifs : “Field Masoned” et “Bordure Indented”.
